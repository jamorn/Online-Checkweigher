วิธีใส่รูป CB500X ของคุณ

ขณะนี้ใช้รูปตัวอย่างจาก Unsplash หากต้องการใช้รูป CB500X จริงของคุณ:

  1. อัพโหลดไฟล์รูปของคุณไปยังโฟลเดอร์เดียวกันกับไฟล์ HTML นี้
  2. แก้ไขโค้ด CSS ในส่วน .analog-clock:
    3. background-image: url('cb500x.jpg'); /* เปลี่ยนเป็นชื่อไฟล์รูปของคุณ */
    background-size: cover;
    background-position: center;
    background-repeat: no-repeat;
    background-color: rgba(15, 23, 42, 0.75); /* ค่านี้ปรับความจางของรูป */
    background-blend-mode: overlay;
  3. ปรับค่า rgba(15, 23, 42, 0.75) โดยเปลี่ยนตัวเลข 0.75 เป็นค่าอื่น (0 = โปร่งใสสุด, 1 = ทึบแสงสุด)

คุณสมบัติทั้งหมดของนาฬิกา

  • นาฬิกาดิจิทัล: แสดงเวลาในรูปแบบ 12 ชั่วโมง พร้อม AM/PM
  • นาฬิกาอนาล็อก: พร้อมพื้นหลัง CB500X ที่ปรับความจางได้
  • ข้อความ "Design by Jamorn": วิ่งวนตามเข็มนาฬิกาในขอบด้านใน
  • Responsive Design: ปรับ layout อัตโนมัติตามขนาดหน้าจอ
  • เปลี่ยนสีธีม: มี 4 สไตล์ให้เลือก
  • แสดงวันที่ไทย: แสดงวันและเดือนเป็นภาษาไทย พร้อมปีพุทธศักราช
  • ตัวเลขหน้าปัด: ปรับตำแหน่งอัตโนมัติตามขนาดหน้าจอ
